# Installation guide

## Run streamlit app

* `$ virtualenv env`
* `$ source env/bin activate`
* `$ pip install -r requirements.txt`
* `$ cd app/`
* `$ streamlit run app.py`

---
## Run jupyter notebooks
* `$ virtualenv env`
* `$ source env/bin activate`
* `$ pip install -r requirements.txt`
* `$ cd notebooks/`
* `$ jupyter notebook`
